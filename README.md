# 99i-Startpage
Simple startpage.
## How to install
```json
1. Download source code.
2. Extract code to one directory.
3. Open chrome://extensions.
4. Enable developer mode (checkbox in right corner).
5. Refresh page.
6. Drag directory with source code to page.
7. Done.
```
